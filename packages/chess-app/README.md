# Bitcoin Computer Chess App

## Usage

Start a Bitcoin Computer Node in the package `node`. Then copy the `.env.example` file.

```
cp .env.example .env
```

Then start the chess server in the package `chess-server` and run the deploy script there.

### Test

```
npm test
```

### Start the App

```
npm start
```
