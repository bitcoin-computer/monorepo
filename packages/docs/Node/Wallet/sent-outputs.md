# sent-outputs

#### `/v1/CHAIN/NETWORK/wallet/:address/sent-outputs`

Returns the outputs that were sent from a given address.

```shell
curl -X GET http://localhost:1031/v1/LTC/regtest/wallet/mkMUZNoiLh4uuuENU5HNZ4Ssxo8BqEQc5t/sent-outputs
```

[
